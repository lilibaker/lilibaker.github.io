Alumni Network

North Broward Preparatory School

About the Project

The Alumni Network for North Broward Prepartory School was having issues disseminating information to alumni post-graudation and wanted to create a way for alumni to cultivate connections. After learning the needs of the organization, I designed a secure webapp that enabled users to search for other alumni, edit their profile, and publish posts about career opportunities. The webapp was presented to the Alumni Network throughout the development process and the feedback received was implemented.

Skills Used

  • Designed test plan, database structure, and user experience maps
  • Created a prototype using Figma

  • Created a NoSQL database with MongoDB
  • Implemented REST API to create career posts and update profile information
  • Authenticated users with Google Authentication through Passport
